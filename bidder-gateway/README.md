# ProcureAI - Bidder Gateway

A high-fidelity bidder portal for the ProcureAI tender evaluation platform, enabling seamless document submission and real-time bid tracking for government procurement hackathon.

## Getting Started

Follow these steps to run the development server locally:

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Run the development server:**
   ```bash
   npm run dev
   ```

3. **Open the application:**
   Navigate to [http://localhost:3000](http://localhost:3000) in your browser to see the result.
